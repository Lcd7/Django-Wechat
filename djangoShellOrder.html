新建项目

django-admin.py startproject project_name

新建app

python manage.py startapp app_name

创建数据库表 或 更改数据库表或字段

python manage.py makemigrations python manage.py migrate

使用开发服务器

python manage.py runserver python manage.py runserver 8001 指定端口

清空数据库

python manage.py flush

创建超级管理员

python manage.p createsuperuser

修改用户密码

pyhton manage.py changepassword username

django项目环境终端

python manage.py shell

数据库数据导入导出

导出： 不指定appname则导出所有的app数据

python manage.py dumpdata [appname] > appname_data.json

导入： 不需要指定appname

导入数据时，models的CharField字段一定要写max_length,不然超出最大长度就会报错 python manage.py loaddata appname_data.json

备注：常用：导入用户数据：

python manage.py dumpdata auth > auth.json
